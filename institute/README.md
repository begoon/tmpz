# Matrix — порты студенческих программ 1995 года

Две учебные программы Александра Дёмина (МАИ, каф. 302, 1995),
написанные на Turbo Pascal 5.5, перенесённые на современный Python.

Обе программы демонстрируют базовые операции над матрицами:
ввод, сравнение на равенство, сложение, умножение на число,
умножение матриц, транспонирование, идентификацию типа
(нулевая, диагональная, единичная, симметрическая,
кососимметрическая, треугольная, вырожденная) и вычисление
определителя.

## Структура репозитория

```text
institute/
├── matrix1/        оригинальные исходники Matrix 1.0 (*.PAS, BGI)
├── matrix1.py      порт Matrix 1.0 на Python + pygame (SDL2)
├── matrix2/        оригинальные исходники Matrix 2.0 (*.PAS)
├── matrix2.py      порт Matrix 2.0 на Python + curses (терминал)
└── README.md       этот файл
```

## Matrix 1.0 — графический (BGI → pygame)

Оригинал рисуется в режиме VGA 640×480 через библиотеку BGI,
работает с мышью и клавиатурой, выводит псевдографические кнопки
с тенями в графическом режиме.

Порт `matrix1.py` повторяет поведение средствами **pygame-ce**
(SDL2) c логическим разрешением 640×480, отмасштабированным в 2×
для современных мониторов. Палитра, шрифты, расположение окон,
анимации и звуковой сигнал максимально приближены к оригиналу.

**Запуск:**

```sh
./matrix1.py
# или
uv run matrix1.py
```

Зависимости: `pygame-ce>=2.5` — объявлены в заголовке uv-скрипта
(PEP 723), отдельной установки не требуется.

## Matrix 2.0 — текстовый (DOS text mode → ncurses)

Оригинал работает в текстовом режиме 80×25, напрямую обращаясь
к видеопамяти B800h, рисуя псевдографические окна, кнопки с
тенями, полосы прокрутки для больших матриц и т.д.

Порт `matrix2.py` повторяет тот же внешний вид средствами
стандартной библиотеки **curses**: синяя боковая панель с
кнопками, тёмно-серое рабочее поле, рамки матриц, подсветка
«горячих» клавиш, анимация операций с настраиваемой задержкой.

**Запуск:**

```sh
./matrix2.py
# или
uv run matrix2.py
```

Зависимости отсутствуют — curses входит в стандартную поставку
Python. Требуется UTF-8 терминал размером не меньше 80×25 с
поддержкой 16 цветов.

### Управление (matrix2)

| Клавиши                  | Действие                          |
|--------------------------|-----------------------------------|
| ↑ / ↓                    | перемещение по меню               |
| Enter                    | выбор пункта                      |
| Esc                      | отмена / выход                    |
| Латинские «горячие»      | V, E, A, N, M, T, I, D, O, X      |
| В редакторе матрицы:     |                                   |
| ← → ↑ ↓                  | перемещение по ячейкам            |
| 0–9, «.», «-»            | ввод числа                        |
| F5 / F6                  | ввод числа строк / столбцов       |
| F7                       | заполнить матрицу                 |
| F10                      | завершить ввод                    |

## Операции

| Пункт меню            | Описание                                           |
|-----------------------|----------------------------------------------------|
| Ввод                  | Ручной ввод матрицы A или B, задание размеров,     |
|                       | заполнение числом или случайными значениями        |
| Равенство             | Поэлементное сравнение A и B                       |
| Сложение              | A + B (требуются совпадающие размерности)          |
| Умножение на число    | Домножение A на скаляр                             |
| Умножение матриц      | A × B (требуется A.cols == B.rows)                 |
| Транспонирование      | Транспонирование A на месте                        |
| Идентификация         | Распознавание типа: нулевая, диагональная,         |
|                       | единичная, симметрическая, кососимметрическая,     |
|                       | треугольная, вырожденная                           |
| Задержка              | Скорость пошаговой анимации (0–200 мс)             |
| О программе           | Информация об авторе                               |
| Выход                 | Завершение работы                                  |

## Требования

* Python 3.10 и новее.
* [`uv`](https://docs.astral.sh/uv/) — для запуска через PEP 723
  shebang (опционально).
* Для `matrix1.py`: любой рабочий стол с поддержкой SDL2.
* Для `matrix2.py`: терминал с UTF-8 и поддержкой минимум 16 цветов.
